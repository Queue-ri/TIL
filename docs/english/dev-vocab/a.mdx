as fuck

<br />

## AFAIK

as far as I know

(확실하진 않지만) 내가 아는 바로는

<br />

## atrocious

흉악한

"누가 이런 문제를 냈냐" 또는 "코드가 너무 엉망이다" 라고 느낄 때 종종 쓰이는 단어

:::info Example

- "Who made this LeetCode problem? Its constraints are **atrocious**."
- "This Java code is **atrocious**... my brain quits."

:::

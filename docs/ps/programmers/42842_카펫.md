### 풀이 코드

- **메모리**: 102000 KB
- **시간**: 2080 ms

:::

```java showLineNumbers
class Solution {
    public int[] solution(int brown, int yellow) {
        int[] ans;
        for (int a = 1; a < 5000; ++a) {
            for (int b = 1; b < 2000000; ++b) {
                if ((2*a + 2*b - 4) == brown && ((a-2)*(b-2) == yellow)) {
                    if (a < b) ans = new int[]{b, a};
                    else ans = new int[]{a, b};
                    return ans;
                } 
            }
        }
        
        return new int[]{-1, -1};
    }
}
```

### 풀이 해설

무지성 완탐 때림 (범위에 sqrt 씌우면 좀 더 괜춘할듯?)

<br />

#### ⚡ 수학적 최적화

지성있는 $O(1)$ 풀이법은 다음과 같다. (싸피로그가 알고 풀이 쓰기에 참 좋음 ㅇㅅㅇ)

<img src="https://velog.velcdn.com/images/qriosity/post/a385bb27-86e4-4781-87bc-a8ba07611f20/image.jpg" width="600px" height="auto" />

```java showLineNumbers title="코드로 옮긴거"
class Solution {
    public int[] solution(int brown, int yellow) {
        int b4 = brown-4;
        int n = (int)(b4 + Math.sqrt(b4*b4-16*yellow)) >> 2;
        int m = yellow / n;
        
        return new int[]{n+2, m+2};
    }
}
```
